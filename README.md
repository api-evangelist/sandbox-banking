# Sandbox Banking (sandbox-banking)
Sandbox Banking provides API integration and connectivity solutions for financial institutions, enabling banks and credit unions to connect with fintech applications through standardized APIs. The platform facilitates data exchange between core banking systems and third-party financial services.
